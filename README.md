## 🚀 Sobre o Projeto
O **MilMatch AI** é uma plataforma de suporte à decisão (DSS) desenvolvida para candidatos a carreiras militares de elite (EsPCEx, AFA, ITA, ESA). O sistema utiliza **Data Science** e **Engenharia de Requisitos** para validar a viabilidade de aprovação, analisando performance física, saúde e projeção financeira.

🔗 **Acesse a Plataforma Viva:** [milmatch-ai.streamlit.app](https://milmatch-ai.streamlit.app/)

## 🛠️ Funcionalidades de Elite
- **Análise de Gap (Radar):** Visualização multidimensional entre o perfil atual e o edital.
- **Dossiê de Saúde:** Checklist exaustivo de exames laboratoriais e radiológicos.
- **Calculadora de TAF Dinâmica:** Índices automáticos baseados em sexo e instituição.
- **Projeção Patrimonial:** Gráficos de juros compostos simulando 30 anos de carreira.
- **Audit Trail:** Estrutura pronta para persistência de dados e auditoria.

## 💻 Tech Stack
- **Linguagem:** Python 3.x
- **Interface:** Streamlit (UI/UX Progressiva)
- **Data Viz:** Plotly (Gráficos Interativos) e Matplotlib
- **Processamento:** Pandas e NumPy
- **Banco de Dados:** SQLModel (ORM)

## 🔧 Como Rodar Localmente
1. Clone o repositório: `git clone https://github.com/seu-usuario/milmatch-ai.git`
2. Instale as dependências: `pip install -r requirements.txt`
3. Execute o app: `streamlit run app_web.py`
